# 评价与作品集系统

完整人格教育不以单次考试作为核心评价，而以长期证据作为评价基础。

## 评价目标

评价不是为了排名，而是为了回答三个问题：

1. 这个人正在成为什么样的人？
2. 这个人已经能创造什么？
3. 这个人下一步应该如何生长？

## 四类证据

### 1. 作品证据

回答：你做出了什么？

包括：

- 产品
- 文章
- 研究报告
- 视频
- 设计
- 活动
- 服务
- 工具
- 艺术作品

### 2. 过程证据

回答：你是如何做到的？

包括：

- 问题定义
- 资料收集
- 方案选择
- 原型迭代
- 失败记录
- 复盘笔记

### 3. 协作证据

回答：你如何与他人一起创造？

包括：

- 角色分工
- 公开承诺
- 同伴评价
- 冲突处理
- 交付可靠性
- 沟通记录

### 4. 人格证据

回答：你的内在系统如何变化？

包括：

- 好奇心轨迹
- 生命力状态
- 专注时长与质量
- 情绪恢复能力
- 责任承担
- 自我修正能力

## 能力评价表

每个维度分为 5 个阶段：

| 阶段 | 描述 |
| --- | --- |
| 1. 被动 | 需要外部推动才能开始 |
| 2. 尝试 | 愿意尝试，但不稳定 |
| 3. 稳定 | 能在普通难度下持续行动 |
| 4. 自驱 | 能主动设定目标并推进 |
| 5. 生成 | 能带动他人、创造方法或形成影响 |

## 作品集结构

每个学习者拥有一个长期作品集，包含：

- 个人成长叙事
- 六维能力雷达图
- 年度代表项目
- 项目详情页
- 失败复盘库
- 合作评价库
- 真实反馈记录
- AI 阶段报告
- 自我声明

## 项目详情页模板

每个项目需要记录：

- 项目名称
- 项目周期
- 参与者
- 问题来源
- 目标对象
- 最终产出
- 使用的知识
- 关键困难
- 迭代过程
- 外部反馈
- 个人复盘
- 相关证据链接

## AI 阶段报告模板

AI 每月生成一份成长报告，但报告必须基于证据。

报告包含：

1. 本月完成了什么
2. 本月最重要的问题是什么
3. 六维能力有什么变化
4. 哪些行为模式值得注意
5. 哪些项目最能代表成长
6. 下月建议关注什么

## 防止评价异化

新系统必须防止作品集变成新的应试包装。

约束：

- 不只看结果，也看过程
- 不只看光鲜作品，也看失败复盘
- 不只看个人能力，也看合作评价
- 不只看效率，也看生命状态
- 不鼓励伪造热爱和表演式成长

评价的目的，是让人更真实地看见自己，而不是制造新的焦虑。
