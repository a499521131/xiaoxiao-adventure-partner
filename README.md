# 小小冒险伙伴

这是一个个人开发者可落地的儿童 AI 伙伴原型项目。

当前阶段不做“替代学校”的完整系统，不做复杂教育平台，不做硬件，不做数字身份，不做大规模试点。现在只做一个能演示、能迭代、能被家长和孩子理解的小产品。

## 当前产品定义

孩子画出一个属于自己的伙伴，系统把它变成统一美术风格的 2D 卡通形象。孩子通过知识小挑战、互动聊天、游戏反馈和回忆记录，让这个伙伴逐步发生变化。

## 当前只保留 6 个落地系统

1. 形象设计系统
2. 知识系统
3. 成长系统
4. 游戏系统
5. 记忆系统
6. 硬件桥接系统

## 第一版只做 5 个页面

1. 创建伙伴
2. 伙伴主页
3. 知识小挑战
4. 互动聊天
5. 回忆故事本

## 当前不做

- 不做原生 App
- 不做真实 3D 自动生成
- 不做复杂骨骼绑定
- 不做完整语音包
- 不做实体硬件
- 不做账号系统
- 不做支付系统
- 不做家长后台
- 不做公开作品集
- 不做数字身份绑定
- 不做真实儿童数据采集
- 不做学校替代方案

## 当前判断

这个项目现在最重要的不是把理念讲完整，而是做出一个小而清晰的演示版本：

孩子创建伙伴，完成一个知识小挑战，伙伴发生故事变化，系统记录一页回忆。
